Research Services Catalog

At San Diego State University, the IT Division offers a wide range of research computing services, available at no cost to students, faculty, and staff. These baseline services include accounts and storage in Google Workspace, access to Zoom, endpoint management software for university-purchased devices, such as laptops, desktops, and iPads, as well as site-licensed software, including SPSS, SAS, MatLab, EndNote, NVivo, Qualtrics, and Globus. All of these services are provided free of charge, allowing the university community to access the tools and resources they need to support their research and academic endeavors.

Schools, departments, labs, and individual investigators with needs beyond those met by baseline (no fee) services can arrange cost recovery for many existing fee-based agreements with the ITD Division.

This Research Services Catalog provides an easy to use tool for explore both free and fee-based services.

For more information, submit a Research and Cyberinfrastructure request in ServiceNow.

Copilot

Microsoft

Microsoft 365 Copilot Chat is your AI assistant for work, grounding Copilot conversations in both the latest large language models (LLM) and information from the web to help you get more done, spark your creativity, and deepen your knowledge.

Gemini

Google

Gemini gives you direct access to Google AI. Get help with writing, planning, learning, and more.

NAIRR

NSF

NAIRR (National AI Resource) is a federally-funded initiative that provides state-of-the-art AI tools, data, and computing infrastructure to accelerate research and innovation across academic, government, and industry sectors.

ChatGPT

OpenAI

ChatGPT is an advanced conversational AI language model developed by OpenAI that generates human-like text responses for a wide range of topics and tasks.

Amazon Web Services

Amazon

Amazon Web Services (AWS) is a comprehensive cloud computing platform provided by Amazon that offers a wide range of services for computing, storage, databases, analytics, machine learning, and more.

Azure

Microsoft

Azure provides a wide range of services, including computing, storage, networking, and artificial intelligence, that can be used to support a variety of use cases, from simple web applications to complex enterprise systems.

Google Cloud Platform

Google

Google Cloud Platform (GCP) is a suite of cloud computing services offered by Google that enables users to build, deploy, and manage applications and services through a global network of data centers.

CloudBank

NSF

CloudBank is an NSF-funded research initiative dedicated to pioneering innovative cloud computing frameworks for the secure, scalable, and efficient management of financial data and digital assets.

TIDE

SDSU IT

Technology Infrastructure for Data Exploration (TIDE) provides access to CPU and GPU for AI/ML workloads, including hosted JupyterHub and AI services.

VERNE

SDSU IT

VERNE is an HPC resource targeting instruction providing CPU and GPU resources. Offering both batch jobs as well as a managed JupyterHub instance.

ACCESS

NSF

ACCESS CI is an NSF project focused on developing scalable, secure, and user-friendly cyberinfrastructure to empower diverse research communities in tackling complex scientific challenges.

CSRC

SDSU COS

The Computational Science Research Center in the College of Sciences provides high-performance computing services through cost recovery processes and the "condo" model.

Globus

Globus

Globus is a high-performance, secure data management service that enables researchers to seamlessly transfer, share, and access large-scale datasets across distributed computing environments.

Research Data Services

SDSU Library

SDSU Research Data Services offers comprehensive support for managing, curating, and preserving research data, empowering scholars with secure and efficient solutions to enhance their research practices.

Google Shared Drive

Google

Google Shared Drive is a service of SDSU's Google Workspaces. Shared Drive allows storage and sharing of data.

Azure Files

Microsoft

Azure Files is a fully managed cloud-based file storage service by Microsoft Azure that offers scalable, secure file shares accessible via standard SMB protocols for seamless integration with applications and workflows.

AWS S3 Storage

Amazon

Amazon S3 is a highly scalable, secure object storage service designed to store and retrieve any amount of data from anywhere on the web.

Virtru

Virtru

Virtru provides easy-to-use data security for information shared through email and integrates with SDSU's Google Workspaces environment. Virtru is intended for project with high compliance requirements including CUI.

Security Consultation

SDSU IT

The IT Security Office (ITSO) provides guidance and recommendations for storing, handling, and using protected data, as well as for complying with common frameworks such as HIPAA, PCI, CUI, PHI, and NIST 800-171, among others.

SAS

SAS Software

SAS software is a comprehensive analytics suite that enables organizations to manage data, perform advanced statistical analyses, and generate actionable business insights.

SPSS

IBM

SPSS software is a powerful statistical analysis tool that streamlines data management and enables users to perform complex analyses and generate insightful visualizations.

MATLAB

MathWorks

MATLAB is a high-performance programming environment designed for numerical computing, data analysis, algorithm development, and visualization across various scientific and engineering disciplines.

REDCap

REDCap

REDCap is a secure, web-based software platform designed for efficient data capture and management in research studies and clinical trials.

Qualtrics

IBM

Qualtrics is a cloud-based experience management platform that enables organizations to design, distribute, and analyze surveys and feedback to drive informed decision-making.

Cloud Server

SDSU IT

Cloud-based virtual machines can provide an alternative to on-premises servers and come in a range of sizes.

Server Hardware

SDSU IT

SDSU's data center can host rack-mounted servers and management services are offered to management operating system and compliance requirements.

Laptop / Desktop

SDSU IT

Desktops and laptops purchased as part of a grant can be procured and deployed, including ensuring compliance, by staff.

Modern Campus CMS

SDSU IT

Modern Campus CMS is operated by the IT Division's Web Services group.

RohanCP

SDSU IT

RohanCP web hosting platform offers simple website hosting or content management with WordPress.
