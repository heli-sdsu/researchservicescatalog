AWS S3 Storage

Overview

 
Amazon S3 (Simple Storage Service) is a cloud-based object storage service that supports scalable, secure, and high-performance data storage for a wide range of use cases. It offers flexible storage classes, built-in security and compliance features, and automatic scaling to accommodate virtually any volume of data. With high durability, availability, and cost-optimization options, Amazon S3 enables efficient data management and access control for diverse workloads. SDSU faculty can contact the university's IT Division for more information on how to utilize the service's storage capabilities.

SDSU's IT Division (ITD) maintains an Amazon Web Services (AWS) enterprise account configured with an AWS landing zone. The landing zone allows ITD to provision new AWS accounts pre-configured with single sign-on (SSO) and secure access.

AWS costs for AWS accounts are cost recovered by ITD on a quarterly basis from either state or foundation funds. A Memorandum of Understanding (MOU) for Cloud Services it executed between ITD and the requesting department or project.

Data Classification & Usage Matrix

Data Type Deidentified Pl1 Pl2 Pl3 HIPAA PHI HIPAA NIST 800-171
Allowed Type? Yes Yes Yes Yes Yes Yes Yes Yes
  • Deidentified - Data stripped of personally identifiable information (PII) cannot be linked back to an individual.
  • PL1 (Protection Level 1) - Restricted or sensitive data requiring strict access controls and encryption.
  • PL2 (Protection Level 2) - Internal-use data requires some access control but has moderate risk if exposed.
  • PL3 (Protection Level 3) - Low-risk data that is publicly available or does not require access controls.
  • HIPAA (Health Insurance Portability and Accountability Act) - Indicates whether the system adheres to HIPAA regulations for storing, processing, and transmitting healthcare data.
  • PHI (Protected Health Information) - Individually identifiable health information that is not regulated under HIPAA.
  • NIST 800-171 - Security requirements for protecting Controlled Unclassified Information (CUI) in non-federal systems.
